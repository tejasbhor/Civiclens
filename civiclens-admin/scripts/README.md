# CivicLens Admin Utility Scripts

This directory contains utility scripts for development and debugging. These scripts are not part of the production build and are intended to be run manually or used for reference.

## debug_officers_api.js

A script to test the Officers API from the browser console.

**Usage:**
1. Open the CivicLens Admin application in your browser.
2. Log in to acquire a valid authentication token.
3. Open the browser's developer console (F12 or right-click -> Inspect -> Console).
4. Copy and paste the contents of `debug_officers_api.js` into the console.
5. Press Enter to run the script. It will fetch officers and log the results.
